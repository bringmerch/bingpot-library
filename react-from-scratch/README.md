# create react app from scratch
> initializer 같은 거 안 쓰고 흥선대원군마냥 빈 폴더에서 리액트 정적앱을 만들어보자 
> 
> 순서가 궁금해질까봐 ㅎ

1. npm init으로 package.json 만들기
2. npm install react react-dom react-scripts 
3. public/index.html 생성 (이름바꾸지마셈)
4. src/index.js 생성
5. package.json에서 type, scripts 항목 수정 
6. src/index.js에 테스트삼아 넣을 src/CustomComponent.js 생성 
7. src/index.js 작성 
8. npm start
