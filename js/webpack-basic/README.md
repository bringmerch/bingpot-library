# Webpack basic usage

## Project Initiation
```console
npm init
-> package.json 생성 

npm install --save-dev webpack webpack-cli
-> webpack, webpack-cli devDependencies로 생성 

npm install lodash
```

## How to build
```console
npx webpack 또는 npm run build
-> dist 폴더에 output 생성 
```

## How to run
```console
IDE에서 index.html 실행 또는 브라우저에서 file:///[로컬 호스트 내 index.html 절대경로] 
```
